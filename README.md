## 📝 Project Description

Smart Attend is an AI-powered system designed to automate attendance using facial recognition and real-time location verification. It improves efficiency, eliminates proxy attendance, and provides accurate attendance tracking with intelligent reporting features.

---

## 🎯 Objectives

* Automate attendance marking
* Prevent proxy attendance
* Improve accuracy and efficiency
* Provide real-time attendance insights
* Enhance user interaction using AI chatbot
